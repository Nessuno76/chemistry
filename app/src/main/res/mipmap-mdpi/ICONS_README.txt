PLACEHOLDER FOR LAUNCHER ICONS

To generate proper launcher icons:
1. Use Android Studio's Image Asset tool:
   - Right-click on res folder
   - New > Image Asset
   - Select Launcher Icons
   - Choose your icon image or create one

2. Or use an online generator like:
   - https://romannurik.github.io/AndroidAssetStudio/
   - https://icon.kitchen/

Required sizes:
- mdpi: 48x48
- hdpi: 72x72
- xhdpi: 96x96
- xxhdpi: 144x144
- xxxhdpi: 192x192

For now, the app will use the default Android icon.
